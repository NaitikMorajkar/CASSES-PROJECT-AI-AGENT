CaseQuery is the world's first SQL-powered murder 
mystery game built on CROO's Agent Commerce Layer.

Players stake real USDC via CAP Protocol, investigate 
real murder cases using SQL queries against a live 
database, and receive automatic payouts when they 
identify the correct killer.

Three original cases:
- The Greenline Double Cross — warehouse theft + 
  murder + twin brothers + mental illness
- The Deshpande Murder — poisoned watch + IPS 
  officer + blackmail + tragic love story  
- The Ghost Protocol — $5M DeFi heist + CROO 
  mixing agent + father murdered by own son

Two gameplay modes:
- SQL Mode — write real queries, maximum reward
- Detective Mode — guided clues, no SQL needed

No luck. No guessing. Pure SQL skill.
Stake. Solve. Collect.

Built on CROO. Powered by CAP. Verified on-chain.
